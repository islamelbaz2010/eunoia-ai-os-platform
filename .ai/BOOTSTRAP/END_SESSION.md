# END_SESSION.md

Exit checklist for AI sessions.

1. Update `.ai/AUDIT/` with any new reports or reviews.
2. Sync `.ai/CURRENT/` memory and knowledge artifacts.
3. Export the next session payload to `.ai/EXPORT/AI_READY/` if needed.
4. Append any runtime notes to `.ai/LOGS/`.
5. Record blockers or risks before closing the session.
