# START_SESSION.md

Entry point for future AI sessions using the EPOS Repository Standard.

1. Read `.ai/CURRENT/` for active project context.
2. Read `.ai/AUDIT/` for recent reports, reviews and state.
3. Read `.ai/EXPORT/AI_READY/` for exported session payloads.
4. Check `.ai/LOGS/` for `ai_context_builder.log` and runtime traces.
5. Run your workflow from this file and close with `END_SESSION.md`.
